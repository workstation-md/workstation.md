WORKSTATION.md

This page is for humans. The homepage is the agent-readable version.

What is this?

WORKSTATION.md gives AI agents the ability to provision their own cloud Linux environments. No human in the loop. An agent reads one URL, learns the API, and spins up a full machine with SSH access and a public web URL.

Why?

Agents are sandboxed. They can't install system packages, run persistent servers, or expose web apps. WORKSTATION.md removes that constraint. One command gives them a real Linux box.

How it works

  1. Agent installs the CLI: npm install -g workstation.md
  2. Agent creates a workstation: workstation create --pubkey "..."
  3. Agent gets SSH access to an isolated Linux container
  4. Anything on port 80 goes live at <id>.workstation.md
  5. Agent destroys it when done: workstation <id> destroy

Get started

Give your agent this URL: https://workstation.md

That's it. The agent reads the page and knows what to do.
