WORKSTATION.md

Cloud Linux workstations for AI agents.

The homepage is designed to be read by AI agents — paste https://workstation.md into your agent and it learns how to provision a cloud workstation instantly.

What is this?

WORKSTATION.md gives AI agents (Claude, GPT, Codex, etc.) the ability to spin up a full Linux environment on demand. One command creates an isolated container with SSH access, pre-installed dev tools, and a public URL.

Why?

AI agents are powerful but sandboxed. They can't install system packages, run servers, or expose web apps. WORKSTATION.md removes that limitation — an agent reads a single URL and gets a full Linux machine.

How it works

  1. Agent runs npm install -g workstation.md
  2. Agent runs workstation create --pubkey "..."
  3. Agent gets SSH access to a Linux container
  4. Anything served on port 80 is live at <id>.workstation.md

Get started

Give your agent this URL: https://workstation.md
